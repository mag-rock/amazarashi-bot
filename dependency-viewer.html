プロジェクト構造の概要

このプロジェクトは以下の主要なディレクトリ構造を持っています：

  • src/application: アプリケーション層のコード
  • src/domain: ドメインロジックを含むコード
  • src/infrastructure: 外部サービスとの連携コード

主要な依存関係

  • Google Cloud Functions Framework
  • Firestore
  • Twitter API
  • Google Spreadsheet API

パッケージ依存関係

このグラフはプロジェクトのnpmパッケージ依存関係を示しています。緑色のノードは実行時依存関係、赤色のノードは開発依存関係です。

ソースコード依存関係

このグラフはプロジェクト内のソースコードファイル間の依存関係を示しています。色分けは以下の通りです：

  • 青色: アプリケーション層
  • 緑色: ドメイン層
  • 赤色: インフラストラクチャ層
  • 黄色: その他
